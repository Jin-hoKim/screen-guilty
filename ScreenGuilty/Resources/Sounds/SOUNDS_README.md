# 사운드 파일

이 디렉토리에 다음 사운드 파일을 추가하세요.

## 필요한 파일

| 파일명 | 설명 | 감정 단계 |
|--------|------|----------|
| sigh.mp3 | 가벼운 한숨 | 살짝 실망 (2~5분) |
| sob.mp3 | 훌쩍거림 | 슬픔/울기 (5~30분) |
| angry.mp3 | "일 안 할 거야?!" | 분노 (30분+) |
| cheer.mp3 | 환호/박수 | 업무 복귀 환호 |
| welcome_back.mp3 | "역시 믿고 있었어!" | 업무 복귀 감동 |

## 무료 효과음 소스

- https://freesound.org
- https://soundsnap.com
- https://mixkit.co/free-sound-effects/

## 형식

- MP3 또는 WAV
- 길이: 1~3초 이내
- 볼륨: -12dB ~ -6dB (너무 크지 않게)
