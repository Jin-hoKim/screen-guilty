# 캐릭터 애니메이션 파일

이 디렉토리에 Lottie JSON 애니메이션 파일을 추가하세요.

## 필요한 파일

| 파일명 | 감정 | 설명 | LottieFiles 검색 키워드 |
|--------|------|------|----------------------|
| peaceful.json | 평화 | 눈 깜빡이며 미소, 루프 | cute character idle, happy face loop |
| disappointed.json | 살짝 실망 | 고개 숙이기, 눈 내리깔기 | sad face, disappointed character |
| sad.json | 슬픔 | 눈물 글썽, 입꼬리 내림 | crying face, teary eyes |
| crying.json | 울기 | 눈물 뚝뚝, 흐느끼기 | crying character, sobbing animation |
| angry.json | 분노 | 얼굴 빨개짐, 연기 뿜기 | angry face, furious character |
| smile.json | 미소 (복귀) | 활짝 웃기 | happy smile, cheerful character |
| excited.json | 환호 (복귀) | 점프 + 박수 + 하트 | celebration, cheering character |

## 에셋 소스

- https://lottiefiles.com (무료 필터 적용)
- 같은 작가의 시리즈 선택 (스타일 통일)
- 배경 투명, 200x200px 이상 권장

## 사용 방법

파일 없을 경우 앱은 자동으로 이모지 기반 폴백 캐릭터를 표시합니다.
